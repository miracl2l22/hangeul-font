미생체 URL: http://webtoon.daum.net/event/misaengfont

* 제주한라산체(Jeju Hallasan)

== Installation ==

1. `/wp-content/plugins/hangeul-web-font`에 플러그인 파일을 업로드 합니다.<br />
Upload the plugin files to the `/wp-content/plugins/hangeul-web-font` directory, or install the plugin through the WordPress plugins screen directly.

2. 플러그인을 환성화합니다.<br />
Activate the plugin through the 'Plugins' screen in WordPress

3. `설정->웹폰트` 에서 사용할 웹폰트를 설정합니다.<br />
Use the `Settings->Hangeul Web Fonts` screen to configure the plugin

4. css에 파일에 `font-family`를 설정해 주세요.<br />
Use `font-family` in stylesheet
